# 周礼生成器

这是一个“古文白话译述体”回复生成器。仓库里包含两版：

- 方案 A：原生 macOS 桌面悬浮助手原型。
- 方案 B：静态 Web 应用，打开浏览器即可使用。

## 方案 B：Web 应用

直接打开 `web/index.html`，或在仓库根目录启动一个静态服务：

```bash
python3 -m http.server 5173
```

然后访问：

```text
http://localhost:5173/web/
```

Web 版支持：

- 粘贴聊天上下文并实时生成回复。
- 对固定模板外的任意聊天内容做兜底分析与生成。
- 输出道德判断、古文依据和可编辑回复。
- 生成回复使用固定句式：“我曾听说，{白话文道理}。这样的道理，您还是好好想想吧。”
- 切换“持平 / 峻切 / 温厚”三种语气。
- 复制古文依据或生成回复。
- 示例切换、清空输入。

当前 Web 版使用本地离线语料索引，不需要运行时联网。语料参考公开古籍文本整理，覆盖仁恕、诚信、义利、自省、礼敬、和同、慎言、明辨、察言、克制、求实、恕道、改过、见贤、忠恕等场景；没有命中具体关键词时，会使用“审慎明辨”路径兜底生成。

参考来源：

- 中国哲学书电子化计划：《论语》 https://ctext.org/analects/zhs
- 中国哲学书电子化计划：《孟子》 https://ctext.org/mengzi/zhs
- 中国哲学书电子化计划：《礼记》 https://ctext.org/liji/zhs

## 方案 A：Mac 原型

这是一个原生 macOS 桌面悬浮助手原型，用于在微信聊天时快速生成“古文白话译述体”回复。

## 第一版能力

- 菜单栏常驻：显示为“周礼”。
- 全局快捷键：`Option + Space` 唤醒。
- 上下文读取：在微信仍是前台时自动读取当前选中的聊天文本；读不到时提示重新选中。
- 回复生成：先判断对方发言违背的道德标准，再匹配古文依据，最后动态生成白话译述风格回复。
- 用户确认：点击“插入聊天框”后，把回复粘贴回唤醒前的应用输入框。
- 安全默认：只插入，不自动发送。

## 古文语料

当前内置语料覆盖《论语》《孟子》《礼记》《中庸》《道德经》《尚书》等经典中的短摘，按仁恕、诚信、礼敬、自省、义利、宽厚、慎言、公正等主题组织。

这些内容来自公共领域经典文本的常见版本方向，适合做本地原型。后续若要继续扩大覆盖面，可以接入维基文库、中国哲学书电子化计划等公开古籍来源，离线整理为结构化语料库，再结合大模型做检索增强生成。

## 构建与运行

```bash
make build
make run
```

如果菜单栏里已经有旧的“周礼”在运行，请先从菜单里点“退出”，再执行 `make run`。macOS 对同一个应用 bundle 可能会复用旧进程，导致新构建没有生效。

`make run` 只打开已构建的应用，不会重新编译。改完代码后使用：

```bash
make rebuild-run
```

构建产物位于：

```bash
.build/ZhouliGenerator.app
```

## 打包分享

生成可分享的 dmg：

```bash
make package
```

产物会生成在 `dist/` 目录，例如：

```bash
dist/ZhouliGenerator-YYYYMMDD-HHMMSS.dmg
```

dmg 内包含 `ZhouliGenerator.app`、`Applications` 快捷入口和安装说明。当前版本使用本地 ad-hoc 签名，适合小范围试用；如果要公开分发，需要使用 Apple Developer ID 签名并公证。

首次运行会请求辅助功能权限。若快捷键、复制或粘贴没有生效，请在：

系统设置 -> 隐私与安全性 -> 辅助功能

中允许“周礼生成器”或当前终端控制电脑。

## 辅助功能权限排查

开发测试时先 `make build`，授权完成后只用 `make run` 启动。macOS 的辅助功能权限会绑定应用的代码身份，反复重新编译或重新签名可能导致系统设置里看起来已开启，但应用内仍判断为未授权。

如果权限状态错乱，按下面步骤重置一次：

```bash
tccutil reset Accessibility local.zhouli.generator
```

然后退出旧的“周礼”，执行 `make build`，再到：

系统设置 -> 隐私与安全性 -> 辅助功能

重新打开 `ZhouliGenerator`。授权后不要再 `make build`，直接 `make run`。

## 微信中使用

1. 在微信中拖选对方的一段发言，不需要手动复制。
2. 按 `Option + Space`。
3. 在悬浮窗里确认或编辑生成结果。
4. 点击“插入聊天框”。

如果提示没有读到选中文本，请先在微信里选中文字后手动按一次 `Command + C`，确认微信本身能复制这段选区；再检查系统辅助功能权限是否已经授予“周礼生成器”。
